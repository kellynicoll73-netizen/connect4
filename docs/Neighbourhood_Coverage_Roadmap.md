# Metro Vancouver — Neighbourhood Coverage Roadmap

**Purpose:** Personal reference for Apt's neighbourhood coverage. Tracks which named
neighbourhoods exist across Metro Vancouver, which ones Apt currently profiles, and
notes for review. Not a project spec — a working checklist for Kelly.

**Scope:** Metro Vancouver (GVRD) only — through Langley and Maple Ridge. Fraser Valley
(Abbotsford, Chilliwack, Mission, Hope) is explicitly out of scope unless we later expand.

**Status:** Draft compilation (June 2026) — needs a long-time-resident review pass to
merge fine-grained sub-areas, drop non-rental areas, and fix any loose placements.

**Legend:** ✅ = profiled in Apt today · ⬜ = not yet · ⭐ = priority rental destination to add next

---

## Summary

- **Covered today:** 25 neighbourhoods
- **Named areas listed below:** ~150 (most are sub-areas; realistically ~40–55 are true rental destinations worth profiling)
- **Biggest near-term gaps (renters):** Brentwood, Central/Upper Lonsdale, Richmond Centre, Guildford/Newton/Fleetwood, Coquitlam Town Centre, Ambleside/Dundarave, Ladner/Tsawwassen, Willoughby Heights

## Suggested completion phases (rough)

1. **Now:** match against the existing 25 (build persona tests on these).
2. **Vancouver depth:** add remaining official Vancouver areas + key informal ones (Fairview, Olympic Village, Gastown, Hastings-Sunrise, Cedar Cottage).
3. **Metro breadth — the ⭐ list:** Brentwood, Lougheed, Central Lonsdale, Richmond Centre, Coquitlam Town Centre, Burquitlam, Guildford, Newton, Fleetwood, New West Quayside, Ambleside, Willoughby Heights, Ladner.
4. **Fill remaining gaps** as demand / testing reveals them.

Target for "complete": roughly **40–55** well-profiled neighbourhoods, vs 25 today.
